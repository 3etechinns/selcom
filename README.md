# Selcom

This gem makes it possible to use the Selcom API without having to deal with XML RPC.
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'selcom'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install selcom
